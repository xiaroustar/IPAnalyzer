# 剪贴板IP地理分析工具

一款自动监控剪贴板中的IP地址并查询地理信息的工具。

轻量版软件（双击即用）及正式版安装软件（正式安装编译）

软件下载地址：
https://ipv4.ink/win/download/

## 功能特点
- 自动监控剪贴板中的IP地址
- 支持IPv4和IPv6地址检测
- 查询IP地理位置和运营商信息
- 桌面通知提醒
- 历史记录保存
- 开机自启动
- 自动更新检测

## 安装方法
1. 下载安装程序 `剪贴板IP地理分析工具_Setup_v1.0.0.exe`
2. 以管理员权限运行安装程序
3. 按照向导完成安装
4. 桌面和开始菜单会创建快捷方式

## 开发者搭建方式
1. 创建根目录\installer，\dist，\build目录
2. 首先执行buildsmall，得到exe轻量版应用宝
3. 下载NSIS工具，将EXE移动到\installer目录下，执行build_installer即可
4. 分发exe软件

## 使用说明
1. 程序启动后会自动最小化到系统托盘
2. 复制任何包含IP地址的文本到剪贴板
3. 程序会自动检测并显示IP地理信息
4. 右键点击托盘图标可以查看更多功能

## 卸载方法
方法1: 控制面板 -> 程序和功能 -> 找到本程序 -> 卸载
方法2: 开始菜单 -> Mingxin Tools -> 剪贴板IP地理分析工具 -> 卸载
